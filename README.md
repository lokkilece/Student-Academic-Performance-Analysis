# Student Academic Performance Analysis

## Project Overview
This project analyzes factors affecting students’ academic performance, including gender, student group, parental education, and test preparation.

## Dataset
- Source: Public student performance dataset
- Records: 1,000 students
- Subjects: Math, Reading, Writing

## Key Questions
- What are the factors affecting students academic performance?
- Do students who complete test preparation score higher?
- How does parental education affect academic performance?
- Does the impact of test preparation vary by parental education level?

## Tools Used
- Microsoft Excel (data cleaning, pivot tables, dashboard)

## Key Insights
- Students who completed test preparation achieved higher average scores across all subjects.
- Higher parental education levels are associated with better academic performance.
- The effect of test preparation is less pronounced among students whose parents hold a master’s degree.
- Based on the findings, schools may consider providing additional test preparation support for targeted student groups to improve overall academic performance.
